# Cloud Infrastructure Monitoring Dashboard

This project is a lightweight Python-based monitoring tool that tracks 
real-time CPU, RAM, and Disk usage. It simulates a cloud infrastructure 
monitoring dashboard used in DevOps and CloudOps.

## Features
- Real-time monitoring of CPU, memory, and disk  
- Logs system metrics in JSON format  
- Runs on Windows, Linux, and macOS  

## Technologies Used
- Python  
- psutil library  
- JSON  

## How to Run
pip install psutil  
python cloud_infra_monitoring_dashboard.py

## Example Output
{"timestamp":"2025-01-05 11:25:10","cpu_usage":12.5,"ram_usage":55.3,"disk_usage":74.1}
